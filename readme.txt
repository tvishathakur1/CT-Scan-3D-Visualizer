This project is written in Python. It needs some packages to be installed. There is a 'requirements.txt' file in the root repository of the code. Please follow the below steps to execute the code:
1. Go to terminal
2. Navigate to the root directory of the folder
3. type - 'pip install -r requirements.txt'
4. This is will download all dependencies
5. Run the main.py code to see results
6. The output is in /output folder of the root directory
